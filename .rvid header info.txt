0x0: "RVID" string
0x4: (u32) File format version
0x8: (u32) Number of frames
0xC: (u8) Frames per second (FPS)
0xD: (u8) Vertical resolution
0xE: (u8) Is interlaced
0xF: (u8) Has audio
0x10: (u16) Audio sample rate
0x12: (u16) Frames are compressed
0x14: (u32) First frame offset
0x18: (u32) Sound stream offset
